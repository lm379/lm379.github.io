第2关：矩形的面积与周长

问题描述
指定矩形的长和宽，计算并输入矩形的面积与周长。

输入
两个用空格分开的整数，分别表示矩形的长和宽。

输出
输出包括两行：
第一行输出矩形的面积，第二行输出矩形的周长。

输入示例
3 4

输出示例
面积是12
周长是14

第3关：逆序数求和

问题描述
输入一个三位正整数，计算并输出这个整数和其逆序数之和。
逆序数是指把原三位数的个位、十位、百位逆序后构成的三位数，例如三位数256的逆序数是652。

输入
一个三位的正整数。

输出
一个等式，按照“原数+逆序数=和”的格式输出。

输入示例
256

输出示例
256+652=908
