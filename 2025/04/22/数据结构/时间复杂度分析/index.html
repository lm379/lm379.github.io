发表于 2025-04-22 分类于 数据结构 阅读次数：
时间复杂度分析，其中直接数循环层数就能看出来的过于简单，本文不讨论

一层循环

一般步骤

  1. 列出 循环次数 t 和 迭代变量 (通常为i,j) 之间的关系
  2. 写出循环终止条件
  3. 联立 1 2 两式解出 t 与 n 之间的关系

例1

首先列出 循环趟数t 和 迭代变量i 之间的关系表，可以得到 i=ti = ti=t 这个关系式

t 0 1 2 3 … t
i 0 1 2 3 … i
